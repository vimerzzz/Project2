B1: Cài đặt npm, nodejs, MySQL Server, MySQL WorkBench
B2: Mở IDE Visual Studio Code
B3: Mở Terminal trong thư mục Project2
B4: Cài đặt các thư viện để chạy chương trình bằng câu lệnh:
    npm i bcryptjs body-parser connect-flash cookie-parser ejs express express-session jquery multer mysql2 nodemon passport passport-local path sequelize validator --save
B5: Vào file params/params.js để chỉnh sửa mật khẩu của MySQL Server được khai báo ở biến password
B6: Mở MySQL WorkBench, kết nối với MySQL Server, và mở file project2-db.sql
B7: Chạy từ dòng 1 cho tới dòng 21

    NOTE:
        Không thể chạy hết file sql vì dữ liệu mật khẩu của tài khoản người dùng bị mã hóa bởi chương trình

B8: Mở lại Terminal để chạy Project bằng câu lệnh:
    npm start
B9: Mở lại file project2-db.sql
B10: Dòng 1 gõ /*
     Dòng 21 gõ */
B11: Chạy toàn bộ file project2-db.sql
B12: Mở Browser, truy cập web app bằng địa chỉ:
    localhost:8080

NOTE:
    Các tài khoản được thiết lập sẵn:
        admin-admin         Role: Admin
        deliver1-deliver1   Role: Deliver
        deliver2-deliver2   Role: Deliver
